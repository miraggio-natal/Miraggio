# Miraggio

## Visão geral
Sistema de gestão de funcionários com portal público para cadastro e painel administrativo para consulta e gestão básica.

## Como executar
1. Entre em [backend-api](backend-api)
2. Instale as dependências com `npm install`
3. Inicie o servidor com `node server.js`
4. Acesse:
   - http://localhost:3000/admin
   - http://localhost:3000/publico
